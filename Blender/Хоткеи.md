Shift+ЛКМ - Движение камеры без вращения
. - приближение к объекту

обжект мод
g - grab - перемещение
r - rotate -вращение
s - scale - масштаб
shift+a - добавить новый объект
shift+d - дубликат

edit mode ( tab - войти)
g - grab - перемещение
r - rotate -вращение
s - scale - масштаб
w - переключаться между режимами выделения
1-3 - выбор выбора - точки, рёбра, грани
shift+alt выбирает группу групп элементов
alt - выбрать всё
alt+a - отменить выбор
ctrl+R - лупкат (разрез) (колесико для увеличение количества разрезов)
ctrl+B - фаска (колесико для скругления)
ctrl+A - all transform перед фаской иногда ( в обжект моде)
e - extrude - выталкивание
i - вставка поверхности

alt+x - прозрачность

ctrl+j - объединить объекты в один

k - knife - нож
	shift - привязка к центру
