### Импорт данных в Obsidian

1. Откройте плагин **Importer** через палитру команд или значок на панели.
2. Выберите файл **.zip** с данными (например, Notion), который вы экспортировали.
3. Включите опцию **Create folders for subpages**, если хотите, чтобы подстраницы были размещены в отдельных папках.
4. Включите опцию **Save parent pages in subfolders**, если хотите сохранить структуру Notion.
5. Нажмите **Import**.

### Базовые плагины

Я бы для начала точно установила бы и использовала следующие плагины:

*Core* (если вдруг еще не включены у вас):
1. **Daily notes** – можно создавать ежедневные заметки для отслеживания мыслей/привычек и т.п.
2. **Bases** - то же, что и базы данных в Notion (но да, есть некоторые отличия)
3. **Page preview** – просмотр содержимого заметки в мини окне при наведении курсора на заметку в тексте (в режиме редактирования надо зажать Ctrl)

*Community* (плагины сообщества)

5. **Calendar** – самый популярный плагин Obsidian, добавляющий календарь для удобной навигации по датированным заметкам
6. **Templater** – расширяет функциональность шаблонов, позволяя запускать JavaScript в вашем хранилище. Позволяет вам автоматизировать процесс создания заметок, объединенных каким-либо контекстом, не переписывая каждый раз одни и те же параметры для каждой новой заметки.
7. **Tasks** – помогает отслеживать задачи по всему хранилищу, поддерживает сроки выполнения и повторяющиеся задачи
8. **Kanban** – позволяет создавать канбан-доски в Obsidian. Подходит для ведения дел.
### Наиболее интересные и важные для меня плагины

1. **Canvas** – базовый плагин, который позволяет визуально упорядочивать заметки, изображения, PDF-файлы, видео, интерактивные веб-страницы, создание mindmap
2. **Auto link title** – удобный плагин, который автоматически вставляет название страницы из браузера при вставке в заметку
3. **Excalidraw** – можно рисовать как хочешь, вставлять, что хочешь. В общем, бомба
4. **Telegram Sync** – быстрое перебрасывание постов из тг в Обсидиан через бот. Отлично подходит, если вам понравился пост в тг, и вы хотите его сохранить
5. **Tracker** – для отслеживания привычек с помощью красивых графиков
6. **Vault statistics** – показывает количество заметок, слов, прикреплений, ссылок
7. **Homepage** – если вы хотите создать базовую страницу со всеми важными для вас заметками
